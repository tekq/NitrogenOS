Discord

Discord is a messaging platform developed by DiscordApp, released in 2015.

-----------------------------------------

To install just do: "lmt --add discord/-ptb/-canary"

Note: just discord will install discord, adding -ptb or -canary to discord will install the PTB or Canary
Average compile time: no compiling needed
