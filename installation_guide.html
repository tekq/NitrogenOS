Installation Guide

There are 2 ways to install Nitrogen

New installation

Download the Arch Linux ISO and burn it to a USB Stick

After you boot of the USB

Clone NitroFI

+-----------------------------------------------------------------------------------+

|curl https://raw.githubusercontent.com/NitrogenLinux/nitrofi/main/nitrofi > nitrofi|

+-----------------------------------------------------------------------------------+

After that is done run it.

+------------+

|bash nitrofi|

+------------+

Existing Arch Linux installation

Clone Nitrogen-ify from GitHub

+-----------------------------------------------------------+

|git clone https://github.com/NitrogenLinux/Nitrogen-ify.git|

+-----------------------------------------------------------+

in the folder that you just cloned run "makepkg", if it asks for any dependencies, install them.

Enjoy your installation!
