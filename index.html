Performance by default, Easily Customized

By default Elements will use the best settings while compiling, but you can easily customize mostly everything about it.

Elements is built on Python, and it is built to be hard to break, Fast, and easy to use.

Simple By Default

Elements is designed to control the entire operating system while keeping it simple with multiple tui interfaces for stuff like containerization.

Infinite Containers, well, while you have enough storage of course.

Innovating, in other ways

Containers and Virtual Machines

Elements has the ability to create containers, so you can play with a VM-like working machine, set up by elements.

Unlike Flatpak and Snaps we don't containerize apps, Elements makes a container to play in and then use later or delete.

Elements is our main tool to create containers and Virtual Machines.
