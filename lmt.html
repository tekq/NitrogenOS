Elements is the official Package Manager for Nitrogen, coded in Python and made to be unbreakable.

Syntax: lmt --command(-c)

Commands examples: --add(-a)/--del(-d)/--help(-h) etc.

Elements compiles all of its software, it also uses its own config, named cfg.py in the /usr/share/elements folder.

Config file coded in Python and automatically used by Nitrogen, the absence of this file will result in an error.
