Neofetch

Neofetch is a cli program developed by KISS Linux creator Dylan Araps to show system info, usually used by Arch users to establish dominance over Ubuntu users.

-----------------------------------------

To install just do: "lmt --add neofetch"

And to run you do "neofetch" in a terminal

Average compile time: >1min
