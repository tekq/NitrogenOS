New Installation

To install it as a new installation you must Download our ISO from the Downloads.

Flash it on a USB Stick using dd and then boot it from your USB Stick.

Existing Arch Installation.

It is possible to convert an Arch Installation to a Nitrogen Installation.

Just clone Nitrogen-ify from GitHub, and run it like a normal PKGBUILD.

Both ways should get you a working Nitrogen Installation with Elements.
